# E-Commerce Returns Analysis
## An Operational Efficiency Study

**Domain:** Operations & Data Analysis
**Type:** Independent BA Case Study
**Tools Used:** Microsoft Excel, Power BI, Google Docs

---

## 📋 Objective
To analyse e-commerce product return patterns, identify key 
drivers of revenue leakage, and propose operational improvements 
to reduce return rate from 17.5% to below 10%.

---

## 📁 Deliverables
- ✅ Returns Analysis Dashboard (Interactive HTML)
- ✅ Returns Data & KPI Framework (Excel)
- ✅ Business Requirements Report
- ✅ Operational Recommendations

---

## 🔍 Key Findings
- Identified 17.5% average return rate vs 10% industry benchmark
- Wrong product delivery accounts for 38% of all returns
- Revenue leakage of ₹9.4M in FY2025
- Peak returns in November due to festive season spike
